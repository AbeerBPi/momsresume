Buisness Requirements

To build an MVP which allows private pilots and student pilots to book aircraft conveniently.

Allow users to see the overview of all aircrafts' booking status.

Ensure that users can filter an aircraft based on its specifications.

Enable the user to set timely notifications.

Problem statement

Websites are difficult for pilots to access on the go. A large overiew of all aircraft available is often presented in a complex way with hard-to-find data. Filtering an airraft by its specifications is not convenient and there are no provisions to send out notifications.

Audience

Student Pilots
Private Pilots

Solutions

Allows user to filter the aircraft based on its specs such as make, engine, and avionics.

App allows user to overview the timing slots of relevant aircraft.

Accessibility

1. Hierarchy
2. Navigation bar with picture icons
3. Use of familiar format of elements. Examples: calendar format, payment screen, notification screen
4. Easy navigation
5. Easily accessible drop down information
6. Font sizes and Font weights
